
Currently only the village_rpg map is set up for the RPG Mode.

Important Commands:

TAB - Opens the inventory.  You can also equip spells.  Use Right Mouse to fire a spell.

F1 - Editmode Toggle

E - Action Key, use to interact with characters and items

/testminimap 1 - Toggle on the minimap

/testproj - shoots a random projectile, try /loop i 50 testproj

/spawn - spawn a character


Tips:
To create you own spells, check out /data/game_rpg.cfg
Characters are created by the /spawn command.
